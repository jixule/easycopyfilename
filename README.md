# easyCopyFileName

## 中文说明

### 功能介绍

这是一个轻量的VS Code插件，帮助你快速复制文件名，提供两种复制模式：

1. **复制文件名**：不带文件后缀
2. **复制文件名(含后缀)**：包含完整文件后缀
3. **打包**：vsce package 打包插件，生成 .vsix 文件

支持在两个场景使用：

- 文件资源管理器右键菜单
- 编辑器顶部标签页右键菜单

### 使用方法

1. 在文件资源管理器中右键点击目标文件，或者在编辑器顶部右键点击打开的文件标签
2. 选择「复制文件名」或「复制文件名(含后缀)」菜单项
3. 文件名会自动复制到系统剪贴板，同时右下角会显示成功提示

### 特性

- 操作简单，无需配置，安装即可使用
- 支持两种常用复制场景
- 同时兼容资源管理器和编辑器标签页
- 插件体积小，运行速度快，无额外依赖

## English Documentation

### Features

This is a lightweight VS Code extension that helps you quickly copy file names, providing two copy modes:

1. **Copy file name**: without file extension
2. **Copy file name (with extension)**: includes full file extension

Available in two scenarios:

- File Explorer context menu
- Editor top tab context menu

### Usage

1. Right-click the target file in the File Explorer, or right-click the open file tab at the top of the editor
2. Select the "Copy file name" or "Copy file name (with extension)" menu item
3. The file name will be automatically copied to the system clipboard, and a success prompt will be displayed in the lower right corner

### Advantages

- Easy to use, no configuration required, ready to use after installation
- Support two common copy scenarios
- Compatible with both Explorer and editor tabs
- Small extension size, fast running speed, no additional dependencies
